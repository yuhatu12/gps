Mapping GPS row data (NMEA format) on real map
---
Arduino + GPSモジュールから取得したGPS生データNMEA形式を実際の地図上へマッピングする．  
テキスト処理およびMappingには，Pythonを使用している．  
また地図データは，(OpenStreetMap)[]から取得し，使用している
